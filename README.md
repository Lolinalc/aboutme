# 🚀 Lolina Digital Solutions

**Desarrollo Web & Marketing Digital** — Sitio web profesional construido con React.

## Tecnologías

- **Frontend**: React 18, React Router 6, CSS custom (sin frameworks)
- **Diseño**: Editorial-premium con tipografía Playfair Display + DM Sans

## Comandos

```bash
# Instalar dependencias
cd client && npm install

# Desarrollo local
cd client && npm start

# Build de producción
cd client && npm run build
```

## Deploy

El proyecto es un frontend estático. Compatible con Vercel, Netlify o cualquier servicio de hosting estático.
